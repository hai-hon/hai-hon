Bảng Tổng Hợp HTML Form – Material Design

Tên Loại Cú pháp / Ví dụ Ý nghĩa / Chức năng
label Element <label for="username">Tên</label> Mô tả input và liên kết với id.
input Element <input type="text" id="u" name="u"> Trường nhập dữ liệu.
select Element <select name="country">...</select> Hộp chọn.
option Element <option value="vn">Việt Nam</option> Một lựa chọn trong select.
textarea Element <textarea name="note"></textarea> Ô nhập văn bản nhiều dòng.
fieldset Element <fieldset>...</fieldset> Nhóm các trường.
legend Element <legend>Thông tin</legend> Tiêu đề cho fieldset.
for Thuộc tính <label for="email">Email</label> Liên kết đến input có id.
type Thuộc tính <input type="text"> Định nghĩa loại input.
id Thuộc tính <input id="username"> Định danh duy nhất.
name Thuộc tính <input name="username"> Tên của biến khi submit.
value Thuộc tính <input value="Giá trị"> Giá trị mặc định / gửi đi.
text Input type <input type="text"> Ô nhập text 1 dòng.
password Input type <input type="password"> Ẩn ký tự khi nhập.
checkbox Input type <input type="checkbox"> Nhiều lựa chọn.
radio Input type <input type="radio" name="gender"> Chọn 1 trong nhiều mục.
submit Input type <input type="submit" value="Gửi"> Nút gửi form.
button Input type <input type="button" value="Click"> Nút bấm thường.
file Input type <input type="file"> Chọn tệp cần upload.
